# CCOP 온톨로지 고도화 로드맵: Phase 2 전략 계획 (Strategic Roadmap)

본 문서는 현재 구축된 V2.0 4-Layer 온톨로지의 강력한 기반 위에서, 미래의 "대규모 데이터 처리"와 "불확실성 제어"를 완벽하게 수행하기 위한 **3대 핵심 고도화 전략(Next-Step Architecture)**을 제시합니다.

---

## 🎯 1. 자산 통제력 강화: 명시적 소유/등록 관계망 (Explicit Ownership)

현재의 연결망이 범인의 "행위(Action)"를 추적하는 것에 집중되어 있다면, 고도화 단계에서는 범인이 숨겨놓은 "자산(Asset)"을 원클릭으로 압류/포착하기 위한 직접 구조를 신설합니다.

*   **배경:** 대포통장과 대포폰의 개통자(명의자)와 실제 사용자가 다를 경우 쿼리가 복잡해집니다.
*   **고도화 전략:**
    *   `registered_to` (기기/계좌 $\rightarrow$ 명의자) 엣지 신설: 통신사 및 은행에서 넘어온 가입자 원부를 다이렉트로 매핑합니다.
    *   `controlled_by` (기기/계좌 $\rightarrow$ 실사용자) 엣지 신설: 통화 기지국 위치나 접속 IP가 특정 인물과 겹칠 경우 AI가 이 엣지를 생성합니다.
*   **기대 효과:** 명의자와 실사용자가 분리된 **'대포 인프라'를 시스템이 스스로 적발**하여 수사관에게 붉은색 노드로 경고를 띄워줍니다. 단 1-Hop 탐색만으로 타겟의 전체 은닉 자산 포트폴리오를 브리핑할 수 있습니다.

---

## 🤖 2. 불확실성 제어: 확률적 식별 및 병합 (Probabilistic Resolution)

사이버 수사 초기에는 "이 IP를 쓴 사람이 저 텔레그램 계정일 것이다"라는 '심증'만 존재합니다. 이 심증을 시스템이 수학적(점수)으로 관리하다가 확증으로 변환시키는 아키텍처입니다.

*   **배경:** 100% 확실한 증거가 나오기 전까지, 파편화된 단서들을 어떻게 그래프에서 관리할 것인가의 문제입니다.
*   **고도화 전략 (Entity Resolution Pipeline):**
    *   **1단계 (심증):** `resolves_to {confidence: 0.7}` 엣지를 가상 자아(`vt_persona`)와 실제 인물(`vt_psn`) 사이에 생성합니다. (수사관 화면에는 70% 확률의 점선으로 표시 유지)
    *   **2단계 (확증 및 병합):** 추가 영장 데이터로 신뢰도가 1.0(100%)을 달성하거나 수사관이 [병합 승인]을 누르면, 시스템 내부에서 두 노드를 **하나의 `vt_psn` 노드로 완전히 Merge(병합)**하고 모든 연결 엣지를 하나로 합칩니다.
*   **기대 효과:** 수사관의 머릿속에만 있던 '가설'을 시스템이 관리(Hypothesis Management)해 주며, 데이터가 쌓일수록 범죄자의 실체가 선명해지는 **'자가 학습형' 수사망**을 구축합니다.

---

## ⚡ 3. 대규모 탐색 최적화: 엣지 기반 역할 인덱싱 (Role Indexing at Edge)

수만 건의 데이터가 쏟아지는 메가급 사건(예: N번방 사건, 거대 폰지사기)에서 DB의 응답 속도를 극대화하는 물리적(Physical) 구조 개편입니다.

*   **배경:** 기존에는 사건(`vt_case`)에서 뻗어 나간 인물들 중 "피의자"만 찾기 위해, 연결된 모든 노드를 열어보고 속성을 검사(Table Scan)해야 했습니다.
*   **고도화 전략:**
    *   단순한 `involves`(연루됨) 엣지를 수사 목적에 맞게 세분화합니다.
    *   $\rightarrow$ `investigates` (사건 $\rightarrow$ 피의자)
    *   $\rightarrow$ `targets` (사건 $\rightarrow$ 피해자)
    *   $\rightarrow$ `witnesses` (사건 $\rightarrow$ 참고인)
*   **기대 효과:** 탐색(Query) 시 노드의 속성을 열어볼 필요 없이 **엣지의 이름(Label)만 보고 길을 찾아갑니다(Pointer Hopping).** 초거대 그래프 데이터베이스 환경에서도 분석 속도가 수십 배 이상 향상되며(Scale-Free 보장), 시스템 부하를 획기적으로 줄입니다.

---

## 🚀 4. 요약 및 추진 로드맵

위 3가지 고도화 전략은 CCOP 시스템을 단순한 **"데이터 시각화 도구"**에서 수사관과 상호작용하며 사건을 능동적으로 풀어가는 **"지능형 사이버 인텔리전스 프로파일러"**로 격상시킵니다.

*   **단기 (Phase 2.1):** 탐색 최적화(역할 인덱싱)를 최우선 적용하여 체감 쿼리 속도 극대화
*   **중기 (Phase 2.2):** 명시적 소유/등록망을 구축하여 '대포 자산' 자동 식별 알고리즘 릴리즈
*   **장기 (Phase 3.0):** 엔티티 병합(Resolution) 파이프라인 완성으로 완벽한 추론형 지식 그래프 달성
