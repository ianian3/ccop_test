# 협력기관 샘플 데이터

이 폴더에는 CCOP 시스템 연동을 위한 샘플 데이터 템플릿이 포함되어 있습니다.

## 파일 목록

| 파일명 | 제출 기관 | 설명 |
|--------|----------|------|
| `금융기관_계좌이체_샘플.csv` | 금융기관 | 계좌이체 내역 |
| `통신사_통화기록_샘플.csv` | 통신사 | 통화기록 데이터 |
| `수사기관_사건정보_샘플.csv` | 수사기관 | 사건 기본정보 |

## 사용 방법

1. 해당 기관 유형에 맞는 템플릿 다운로드
2. 템플릿 형식에 맞게 데이터 입력
3. UTF-8 인코딩으로 저장
4. CCOP 시스템에 업로드 또는 API 전송

## 주의사항

- 모든 파일은 **UTF-8** 인코딩으로 저장
- 날짜/시간 형식: `YYYY-MM-DD HH:MM:SS`
- 금액: 원 단위, 소수점 없이
- 전화번호: `XXX-XXXX-XXXX` 형식
