

1.1 데이터 표준 적용 규칙
테이블명: 모든 테이블은 TB_ 접두어를 사용하며, 영문 대문자로 표기합니다.
컬럼명: 표준 용어 사전에 등록된 영문 약어를 우선 사용합니다. (예: 일련번호 → SN, 일시 → DT 등)
데이터 타입:
식별자(ID, 번호 등): VARCHAR (가변 길이 문자열)
일시(Date/Time): TIMESTAMP (PostgreSQL 표준) 또는 CHAR(14)
수치: NUMERIC
여부: CHAR(1) ('Y'/'N')
긴 텍스트: TEXT (PostgreSQL 최적화)
제약조건: 각 테이블의 PK(Primary Key)는 _PK 접미사를 사용하여 명시합니다.

A. 사건/관리 도메인 (Case & Control)
-- 1. 사건 마스터 (TB_INCDNT_MST)
CREATE TABLE TB_INCDNT_MST (
    INCDNT_NO       VARCHAR(20)     NOT NULL, -- 사건번호
    INCDNT_NM       VARCHAR(300)    NOT NULL, -- 사건명
    INCDNT_TYP_CD   VARCHAR(6)      NULL,     -- 사건유형코드

COMMENT ON TABLE TB_INCDNT_MST IS '사건 마스터';
