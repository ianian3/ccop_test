## 6. 디지털증거 해시값 표준

### 6.1 해시 알고리즘 (HASH_ALG)

| 코드 | 알고리즘 | 길이(bit) | 길이(hex) | 권장 여부 |
|------|----------|----------|----------|----------|
| MD5 | MD5 | 128 | 32 | ❌ 사용금지 |
| SHA1 | SHA-1 | 160 | 40 | ⚠️ 권장안함 |
| **SHA256** | **SHA-256** | **256** | **64** | **✅ 필수** |
| SHA384 | SHA-384 | 384 | 96 | ✅ 권장 |
| SHA512 | SHA-512 | 512 | 128 | ✅ 권장 |

> **⚠️ 중요**: 대검찰청 디지털증거 수집 기준에 따라 **SHA-256 이상** 사용 필수

### 6.2 해시값 속성 구조

| 속성명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| `hash_alg` | VARCHAR(10) | ✅ | 알고리즘 (SHA256) |
| `hash_val` | VARCHAR(128) | ✅ | 해시값 (hex) |
| `hash_dt` | TIMESTAMPTZ | ✅ | 해시 생성일시 |
| `hash_by` | VARCHAR(50) | ✅ | 생성자/도구 |

### 6.3 해시값 생성 예시

```python
import hashlib

def calculate_hash(file_path: str, algorithm: str = 'sha256') -> dict:
    """디지털증거 해시값 계산"""
    hash_obj = hashlib.new(algorithm)
    
    with open(file_path, 'rb') as f:
        for chunk in iter(lambda: f.read(8192), b''):
            hash_obj.update(chunk)
    
    return {
        'hash_alg': algorithm.upper(),
        'hash_val': hash_obj.hexdigest(),
        'file_size': os.path.getsize(file_path)
    }

# 사용 예시
result = calculate_hash('/evidence/suspect_phone.dd')
# {'hash_alg': 'SHA256', 'hash_val': 'a1b2c3d4...', 'file_size': 123456789}
```

### 6.4 그래프 노드에 해시값 저장

```cypher
CREATE (:vt_file {

```
